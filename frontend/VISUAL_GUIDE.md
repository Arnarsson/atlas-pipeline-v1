## Color Scheme

### Primary Colors
- **Blue** (Primary): Trust, reliability, data
  - Headers, buttons, active states
  - Progress bars, links

### Status Colors
- **Green**: Success, passed, compliant
  - Quality checks passed
  - Completed runs
  - High confidence PII

- **Yellow**: Warning, attention needed
  - Medium confidence PII
  - Compliance warnings
  - Performance alerts

- **Red**: Error, failed, critical
  - Quality checks failed
  - Failed pipeline runs
  - Critical compliance issues

- **Purple**: PII, sensitive data
  - PII badges
  - Sensitive data indicators

### UI Elements
- **White**: Cards, backgrounds
- **Gray**: Text, borders, disabled states
- **Gradient**: Hero sections, featured cards
